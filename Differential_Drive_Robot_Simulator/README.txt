 ======================================
  ROBOT AND ENVIRONMENT INITIAL VALUES
 ======================================

 Environment properties; map (window) dimensions can be changed from the top of the code (global variables).
   > map_width
   > map_height

 Robot initial properties; x, y, theta (pose) can be changed from the top of the code (global variables).
   > start_x
   > start_y
   > start_theta

 
 ====================
  ROBOT CONTROL KEYS
 ====================

 4 -> increment left wheel velocity (Vl)
 1 -> decrement left wheel velocity (Vl)
 6 -> increment right wheel velocity (Vr)
 3 -> decrement right wheel velocity (Vr)
 8 -> increment right and left wheel velocity
 2 -> set right and left wheel velocity to 0
 5 -> set the wheel velocities to the value of the lowest wheel velocity
